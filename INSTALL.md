# Frank Media - Installation Instructions

## Quick Setup

The project has been redesigned with a modern, professional UI using TailwindCSS and shadcn/ui components. To complete the installation, please run these commands in your terminal:

### Frontend Setup
```bash
cd frontend
npm install
npm run dev
```

### Backend Setup (in separate terminal)
```bash
cd backend
npm install
npm run dev
```

### Or run both together
```bash
npm run dev
```
